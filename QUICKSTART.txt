🚀 QUICK START (3 STEPS):

  1. Install packages:
     pip install -r requirements.txt

  2. Setup FormSpree (for email):
     - Go to: https://formspree.io
     - Sign up (FREE)
     - Create form & get ID
     - Edit templates/feedback.html
     - Replace YOUR_FORMSPREE_ID with your ID

  3. Run app:
     python app.py
     Open: http://127.0.0.1:5000
